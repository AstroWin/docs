    Install Windows

    Welcome to the Windows Installation guide. AstroOS currently supports only Windows 11.

    System Requirements

    Before you start, ensure your system meets the following minimum requirements:

    • Processor: 1 GHz or faster with at least two cores on a compatible 64-bit processor.
    • RAM: 4 GB or more.
    • Storage: 64 GB or larger storage device.
    • Graphics: DirectX 12 compatible graphics / WDDM 2.x.
    • Display: >9‚Äù with HD Resolution (720p).
    • Internet Connection: Internet connectivity is necessary to perform updates and download and take advantage of some features.

    Preparing for Installation

    1. Backup Your Data: Ensure all important data is backed up to an external drive or cloud storage.
    2. Download Windows 11 ISO: Obtain the latest version of the Windows 11 ISO file from here.

    Creating a Bootable USB Drive

    To install Windows 11, you need to create a bootable USB drive. Follow these steps:

    Using Rufus (Windows)

    1. Download and install Rufus.
    2. Insert a USB drive (8 GB or larger) into your computer.
    3. Open Rufus.
    4. Under Device, select your USB drive.
    5. Click Select and browse to the location of the downloaded Windows 11 ISO file.
    6. Ensure the following settings:
    7. Partition scheme: GPT
    8. Target system: UEFI (non-CSM)
    9. Click Start and wait for the process to complete.

    Using Balena Etcher (Windows/Mac/Linux)

    1. Download and install Balena Etcher.
    2. Insert a USB drive (8 GB or larger) into your computer.
    3. Open Balena Etcher.
    4. Click Flash from file and select the Windows 11 ISO file.
    5. Select the USB drive.
    6. Click Flash and wait for the process to complete.

    Installing Windows 11

    1. Insert Bootable USB Drive: Insert the bootable USB drive into the target computer.
    2. Boot from USB:
    3. Restart your computer.
    4. Enter the BIOS/UEFI settings (usually by pressing a key such as F2, F12, Delete, or Esc during startup).
    5. Change the boot order to prioritize the USB drive.
    6. Begin Installation:
    7. Save the BIOS/UEFI settings and restart.
    8. The computer should boot from the USB drive and display the Windows 11 installation screen.
    9. Follow On-Screen Instructions:
    10. Select your language, time, and keyboard preferences, then click Next.
    11. Click Install Now.
    12. Enter your product key or choose to enter it later.
    13. Accept the license terms and click Next.
    14. Choose Custom Install.
    15. Select the partition where you want to install Windows 11. You can delete existing partitions to create new ones if necessary. Click Next.
    16. Installation Process: The installation process will begin. This may take some time. Your computer may restart several times.

    Windows OOBE

    When the installation finishes you need to complete an another setup called OOBE.

    1. Bypass Network
    2. Disconnect your computer from the Internet.
    3. Press Shift + F10 to open Command Prompt.
    4. Type oobe\bypassnro and click enter.

    5. Your PC will restart.

    6. Finish the setup
